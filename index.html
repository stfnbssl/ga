Overview Strapi db schema Json schemas

Questo documento sintetizza il modello dei dati di Gestione Asset.
In questa pagina abbiamo un diagramma dei contentTypes di Strapi (database) con evidenziati gli attributi JSON che contengono configurazioni e metadati.

Le linee (mancano le frecce verso destra) mostrano, a livello macro, come i dati di configurazione e i metadati confluiscono nel JSON dell'ingestionConfig.

I nomi dei contentType e degli attributi JSON sono cliccabili.

I link conducono al dettaglio della struttura dati.

Per quanto riguarda i contentType il dettaglio contiene descrizioni, note e richieste di chiarimento che dovreste verificare (Alessio).

Per il JSON dell'ingestionConfig una verifica di dettaglio dovrà avvenire credo quando avremo un prototipo funzionante.

Per il JSON dei comandi da quel che ho capito sarà un parziale dell'ingestionConfig e sul contenuto preciso mi fornirete le specifiche. La api sarà similare a quella per produrre l'ingestionConfig.
